# plantpopnet
Code tools for working with PlantPopNet data

## Plot_mapper.R
Makes map tagged census plants

```R
map_plla(s = "STR", t = "T1", p = "P1")  # makes map of plants in Angelo Coast Range, transect 1, plot 11
map_plla(s = "STR", t = 1, p = 1)  # same as above
```
